# 🎵 Spotify-Style Music Player — Day 10 of #10DaysOfWebDev

The grand finale project! A fully functional music player with play, pause, skip, progress bar, volume control, and a clickable playlist — built completely with HTML, CSS, and JavaScript.

---



## ✨ Features

- Play / Pause toggle button
- Next & Previous song controls
- Draggable progress bar with live time updates
- Volume control slider
- Clickable playlist with active song highlight
- Auto-plays the next song when one ends
- Clean Spotify-inspired dark UI
- Fully responsive design

---

## 🛠️ Built With

- HTML
- CSS
- JavaScript (Vanilla)
- HTML5 `<audio>` element

---

## 📁 Project Structure

```
music-player/
├── index.html
├── style.css
├── script.js
├── images/
│   └── (album art images)
└── songs/
    └── (mp3 files)
```

---

## 🚀 How to Run

1. Clone or download the files
2. Add your own song files inside the `songs/` folder and album art inside the `images/` folder
3. Open `index.html` in your browser
4. Click play and enjoy! 🎶

---

## 💡 What I Learned

- Working with the HTML5 `audio` element
- Updating the UI in real time using the `timeupdate` event
- Managing player state (current song, play/pause) in JavaScript
- Building a dynamic playlist from an array of song objects
- Converting seconds into minutes:seconds format
- Highlighting the active song in a list dynamically
